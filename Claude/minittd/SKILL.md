# 最小可验证变更（Roblox · Claude 版）

## 工作流
1. 先完成任务分类。
2. 选择验证级别并定义验证方法。
3. 写编码前行为声明。
4. 需要可测试时先做"红"（失败测试或重现）。
5. 只做最小"绿"实现。
6. 执行验证并给出结果。
7. 仅在必要时做受限重构。

---

## 1) 任务分类（先做）

在编码前明确类型：

| 类型 | 示例 |
|------|------|
| 逻辑 / 行为 | 伤害计算、状态机、碰撞判定 |
| 错误修复 | 已知现象的可复现 bug |
| 重构 | 去重、局部清理 |
| UI / 表现层 | 界面显示、动画调整 |
| 配置 / 内容 | 数值表、道具定义 |
| 原型 / 试验 | 快速验证想法 |

---

## 2) 选择验证级别

### 严格（测试优先，必须）
适用：核心逻辑、状态机、伤害/计算系统、可稳定复现的 bug

工具链：
- `mcp__Roblox_Studio__execute_luau` 写最小重现脚本
- `mcp__Roblox_Studio__get_console_output` 捕获运行结果
- 实现前必须确认失败且失败原因正确

### 轻度（推荐验证）
适用：集成密集逻辑、事件驱动系统、客户端-服务端交互

工具链：
- `mcp__Roblox_Studio__start_stop_play` 触发游戏流程
- `mcp__Roblox_Studio__get_console_output` 观察日志与状态
- 测试不自然时可不写测试，但必须有可执行验证步骤

### 手动验证（可跳过严格测试）
适用：UI/视觉、内容/配置、一次性脚本

规则：
- 必须给出可执行手动检查清单

---

## 3) 编码前行为声明（必须）

开始编码前先写清楚：
- **添加/修复**：具体行为是什么
- **不得改变**：哪些现有行为不能受影响
- **成功标准**：怎样算完成
- **验证级别**：严格 / 轻度 / 手动

---

## 4) 编码前探索（按需）

用以下工具快速定位相关代码，避免盲改：

```
mcp__Roblox_Studio__script_grep      -- 搜索关键词/函数名
mcp__Roblox_Studio__script_read      -- 读取具体脚本内容
mcp__Roblox_Studio__search_game_tree -- 查找实例层级
mcp__Roblox_Studio__inspect_instance -- 查看实例属性与子级
```

---

## 5) 红（仅适用可测试任务）

- 用 `mcp__Roblox_Studio__execute_luau` 写一个最小失败重现
- 用 `mcp__Roblox_Studio__get_console_output` 确认失败且原因正确
- 在此之前不写生产代码

---

## 6) 绿（最小实现）

用 `mcp__Roblox_Studio__multi_edit` 编辑脚本，只写满足目标行为的最小代码。

硬规则：
- 不做推测性抽象
- 不加无需求选项/配置
- 不加重试/回退/额外守卫（除非已证明必需）
- 不扩散到无关范围

---

## 7) 验证（必须）

| 验证级别 | 步骤 |
|---------|------|
| 严格 | `execute_luau` 跑重现脚本 → `get_console_output` 确认通过 |
| 轻度 | `start_stop_play` 触发流程 → `get_console_output` 观察预期日志/状态 |
| 手动 | 对照手动 checklist 逐项确认 |

游戏内验证（严格/轻度）的运行时长不得低于 8 秒，再采集 `get_console_output`。

验证应针对行为结果，而非实现细节。

---

## 8) 重构（可选，且受限）

仅在以下条件都满足时允许：
- 行为已验证
- 变更局部且小
- 仅提升清晰度或去重

禁止：大规模重组、跨文件重构、未请求的公共接口变更。

---

## 硬性禁止项

- **防御性过度设计**：无证据不加回退、重试、恢复路径
- **范围蔓延**：不修无关问题，不"顺手优化"
- **虚假严谨**：不写无意义测试，不测实现细节
- **过早抽象**：无当前需求不泛化
- **静默行为变更**：不得改变无关逻辑或引入隐性副作用

---

## Philosophy

- 正确 > 聪明
- 简单 > 灵活
- 已验证 > 假设成立
- 最小 > 完整

---

## 不适用场景

以下场景不要使用本 skill：
- 头脑风暴
- 系统设计
- 规格文档编写
- UI 资产生成
- 纯内容创作
