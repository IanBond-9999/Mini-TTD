# Mini-TTD: 为 Roblox 项目设计的轻度 TTD 开发流程

**在 Roblox 项目中使用 TTD 流程，目的在解决 AI 产出不稳定**

**适用范围**
- 新项目开发核心战斗底层框架
- 开发核心、重型系统
- 开发系统模块中的功能
- 开发技能、道具、玩法规则等功能机制
- 修复已知现象 BUG

**不适用**
- 做 UI
- 建模
- 一次性脚本
- 明确的内容型脚本，例如列出 300 个国家的国旗编号这类固定的内容脚本

---

**Codex 版本**
- 存放地址【C:\Users\你电脑用户的名称\.codex\skills】
- 使用方式【在对话中输入 /minittd 即可调出】
