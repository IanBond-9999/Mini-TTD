# Mini-TTD: 为 Roblox 项目设计的轻度 TTD 开发流程

**在 Roblox 项目中使用 TTD 流程，目的在解决 AI 产出不稳定**

**适用范围**
- 新项目开发核心战斗底层框架
- 开发核心、重型系统
- 开发系统模块中的功能
- 开发技能、道具、玩法规则等功能机制
- 修复已知现象 BUG

**哲学**
- 正确 > 聪明
- 简单 > 灵活
- 已验证 > 假设成立
- 最小 > 完整

**不适用**
- 头脑风暴
- 系统设计
- 规格文档编写
- UI 资产生成
- 纯内容创作

---

- 使用方式【在对话中输入 /minittd 即可调出】
